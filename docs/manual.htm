Sample User Manual

Welcome

There is no need for a separate User Manual for the Authoring Kit Sample data files for Hero Lab®. The Sample game system is built upon the Skeleton data files that provide the starting framework for creating your own custom data files. Documentation on the Skeleton data files is provided within the Authoring Kit wiki (and PDF) that can be found at http://hlkitwiki.wolflair.com. A complete walk-through showing the detailed creation of the Savage Worlds data files is also provided on the wiki. Consequently, separate documentation here would be redundant.

Legal Information

Custom Output DTD

If your data files provide custom output, you should provide a DTD for users to write their XSL files against. You can find the sample game system DTD here.
